# Data Folder

This folder contains the initial dataset used for the capstone prototype.

## Files

- `mental_health_literature.csv`: Initial dataset of mental health literature records used by the baseline pipeline.
- `sample_papers.csv`: Smaller sample file used for testing and demonstration.

## Notes

The current dataset is intended for prototype development and workflow testing. The next phase of the project will expand the dataset using public literature databases such as PubMed. The goal is to create a more reliable and reproducible dataset for the final analysis.
