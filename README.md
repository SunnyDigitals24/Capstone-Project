# Mental Health Literature Review AI Assistant

## DASC 690 Data Science Capstone Project

### Project Title
**Evaluating Agent-Based AI Tools for Supporting Mental Health Literature Reviews**

### Research Question
**How can agent-based AI tools help organize and summarize mental health research literature more efficiently?**

---

## Project Overview

This repository contains the source code and supporting materials for my DASC 690 Data Science Capstone Project at Old Dominion University.

The goal of this project is to investigate how artificial intelligence, specifically agent-based AI systems and Large Language Models (LLMs), can support researchers during the literature review process. As the amount of published mental health research continues to increase, conducting comprehensive literature reviews becomes increasingly time-consuming. This project explores whether AI-assisted workflows can improve the efficiency of organizing, summarizing, and synthesizing research articles while maintaining the quality and relevance of the extracted information.

The project focuses on mental health literature related to depression, anxiety, sleep, and overall psychological well-being.

---

# Project Objectives

- Investigate the use of AI-assisted workflows for literature reviews.
- Develop a baseline pipeline for processing research articles.
- Generate concise summaries from mental health literature.
- Extract important keywords and topics.
- Evaluate the quality of generated summaries.
- Compare baseline methods with future LLM-based approaches.

# Methodology

The current workflow consists of the following steps:

1. Collect and organize mental health research articles.
2. Preprocess article text for analysis.
3. Generate baseline summaries.
4. Extract keywords and important topics.
5. Evaluate generated summaries using quantitative metrics.
6. Export processed results for further analysis and visualization.

This baseline workflow provides a foundation for future integration of LLM-based summarization methods and agent-based AI workflows.

---

# Preliminary Results

Initial testing demonstrates that the processing pipeline successfully loads research articles, generates baseline summaries, extracts keywords, and exports structured output files. These preliminary results establish a working foundation for comparing traditional summarization methods with future AI-assisted approaches.

Although evaluation is still ongoing, the current implementation confirms that the workflow functions as intended and provides a framework for future experimentation.
